About

I played a key role in developing the Scoring System for the 13th PaDaeng National Angklung Music Competition, focusing on redesigning the assessment system to be more efficient, accurate, and accessible online. My contributions included:

  • Developing a digital scoring system to replace the manual assessment process.
  • Ensuring accurate evaluations by implementing methods to eliminate extreme scores.
  • Enhancing transparency through standardized ranking calculations.
  • Providing data-driven insights to support chief organizers and the advisory board in making well-informed decisions.
I successfully completed this format and it was used in the competition.

Context

Lomba Musik Angklung PaDaeng (LMAP) is a biennial national competition organized by Student Activity Units Keluarga Besar Bumi Siliwangi. Held at Universitas Pendidikan Indonesia (UPI), this event provides a platform for participants across various levels, including:

  • Kindergarten
  • Elementary school
  • Junior high school
  • Senior high school
  • University students
  • General public

Problem

The existing scoring system relied on a manual assessment method, where judges submitted scores separately. This approach led to several key issues:

  • Calculation errors due to manual input and human mistakes.
  • Difficulty in handling extreme scores, which affected fairness.
  • Ties in participant rankings, making it hard to determine winners.
  • Inefficient offline processing, causing delays in result announcements.

Business Problem

To improve accuracy and efficiency in the judging process, the committee needed a scoring system that could:

  • Compare three assessment methods to determine the fairest approach:
    1.  Summing all scores from each judge.
    2.  Eliminating the highest and lowest scores.
    3.  Calculating ranking scores from each judge.
  • Introduce standardization features to prevent identical participant scores.
  • Transition the assessment format from offline to an accessible online system.
  • Refine the previous scoring formula to improve clarity and usability.
  • Automate the reporting process, allowing the system to generate:
    • Legger (score summary).
    • Report cards.
    • Champion certificates.

Processes & Considerations

The dataset contains important information regarding:

  • Participant score data in Input Sheet
  • Standardized score conversion in Conversion Sheet
  • Recap of participant scores in Legger Sheet
  • Final results and rankings in Raport Sheet
  • Overall rankings, including winners (1st-3rd), consolation (1st-2nd), and best conductor/vocalist in the Champion Sheet

STEP 1

DATA PROCESSING IN INPUT SHEET

To ensure accurate results, it is important to clean and format the data properly. The following steps should be taken:

  • Remove rows with empty values.
  • Adjust the data format by converting mismatched data types.

Apply color formatting in Input Sheet to highlight problematic values in red if they meet any of these conditions:

  • Value is empty
  • Value is 0
  • Value is less than 30
  • Contains letters

Additionally, an indicator should be added to ensure jury input completion, verifying that all necessary data has been entered correctly.

STEP 2

STANDARDIZATION & EXTREME VALUE ELIMINATION

To ensure fairness and consistency in scoring, a standardization process is applied to normalize the scores and eliminate extreme values.

  • Convert scores using a predefined conversion standard.
  • Identify and remove outlier values that could distort the final ranking.
  • Ensure that extreme values do not unfairly influence the overall score by excluding the highest and lowest scores from the calculation.

STEP 3

PREPARING LEGGER FOR RAPORT

The Legger Sheet is prepared as a recap that will be used in the Raport Sheet. The values have been filtered by removing the highest and lowest scores to make the final score more representative and unbiased.

STEP 4

CREATING AN INTEGRATED RAPORT

The Raport Sheet is built with automatic integration using lookup functions to retrieve data from the Legger Sheet. This allows for dynamic data retrieval, ensuring that each participant's results can be viewed quickly without manual input.

Additionally, the final rankings for 1st-3rd place and 1st-2nd consolation winners are summarized for easier evaluation of the top participants.

STEP 5

FINAL RESULTS & VERIFICATION

The final step involves systematically organizing the results with jury signatures as validation proof. All processed results will be displayed in the Raport Sheet with a well-structured layout that automatically updates when participant data changes.

Furthermore, the final rankings will be compiled in the Champion Sheet, including:

  • Winners (1st, 2nd, and 3rd place)
  • Consolation winners (1st and 2nd place)
  • Best Conductor/Vocalist

By following these steps, the scoring system becomes more structured, automated, and easily accessible.

Result

The implementation of the new Scoring System for the 13th PaDaeng National Angklung Music Competition successfully addressed the key challenges of the manual assessment method. The system provided significant improvements in:

  • Eliminating Calculation Errors & Score Discrepancies: Automated processes reduced human error and ensured accurate rankings.
  • Ensuring Fairness: The removal of extreme scores minimized bias and improved objectivity.
  • Preventing Score Manipulation & Bribery: Standardized calculations made unfair score alterations impossible.
  • Enhancing Transparency: Judges and organizers could easily interpret results without ambiguity.
  • Increasing Efficiency: The shift to an online system expedited data processing and result announcements.
  • Automating Report Generation: Scores, rankings, and certificates were generated instantly, reducing administrative workload.

By integrating this automated and standardized scoring system, the competition achieved a faster, more reliable, and corruption-free assessment process, ensuring that every participant was judged fairly based on their actual performance.
